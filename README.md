1. Планування
Мета: Створити мобільний застосунок для формування корисних звичок. Додаток має допомагати користувачу не забувати про щоденні ритуали (спорт, вода, читання) та мотивувати його через візуалізацію прогресу.
Основна функція: Користувач повинен мати можливість бачити список своїх звичок на сьогодні та відмічати їх виконання одним натисканням.
2. Аналіз вимог
Як користувач, я хочу створювати нову звичку, щоб почати відстежувати свій прогрес у новій справі. ✅ (Must have)

Як користувач, я хочу відмічати звичку як виконану, щоб бачити свій успіх за день. ✅ (Must have)

Як користувач, я хочу видаляти звички, щоб мій список залишався актуальним.

Як користувач, я хочу бачити відсоток виконання за день, щоб розуміти свою ефективність.

Як користувач, я хочу редагувати назву звички, якщо я припустився помилки.

5. Тестування
Тест додавання: Вводимо назву "Йога" -> Натискаємо "Зберегти" -> У списку з'явився новий пункт "Йога".

Тест виконання: Натискаємо на чекбокс -> Колір пункту змінюється на зелений, лічильник прогресу збільшується.

Тест на пусті дані: Натискаємо "Зберегти", не ввівши назву -> Система видає помилку "Назва не може бути порожньою".
